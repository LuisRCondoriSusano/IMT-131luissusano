# 📘 Programación Superior

## 📌 Descripción de la Materia
La materia **Programación Superior** tiene como objetivo profundizar en los fundamentos de la programación, aplicando conceptos avanzados para el desarrollo de software estructurado, modular y eficiente.

Se enfoca en el uso de buenas prácticas, estructuras de datos, programación orientada a objetos y manejo del sistema de archivos, utilizando herramientas de desarrollo en entorno Linux.

---

## 🎯 Objetivos

- Comprender la diferencia entre lenguajes compilados e interpretados.
- Aplicar principios de Programación Orientada a Objetos (POO).
- Manejar estructuras de datos básicas y avanzadas.
- Utilizar correctamente el sistema de archivos en Linux desde la terminal.
- Implementar buenas prácticas de codificación y documentación.
- Trabajar con control de versiones utilizando Git.

---

## 🧠 Contenidos Principales

### 1️⃣ Fundamentos Avanzados
- Tipos de datos
- Estructuras de control
- Funciones y modularidad
- Manejo de archivos

### 2️⃣ Programación Orientada a Objetos
- Clases y objetos
- Encapsulamiento
- Herencia
- Polimorfismo
- Abstracción

### 3️⃣ Estructuras de Datos
- Arreglos
- Listas
- Pilas
- Colas
- Manejo de memoria

### 4️⃣ Entorno Linux
- Navegación por el sistema de archivos
- Permisos (usuario, grupo y otros)
- Comandos básicos:
  - `ls`
  - `cd`
  - `chmod`
  - `cp`
  - `mv`
  - `rm`

---

## 🛠️ Tecnologías Utilizadas

- Lenguaje C / C++
- Linux (Ubuntu)
- Git y GitHub
- Terminal Bash
